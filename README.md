## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   PORT=3000 npm start
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

4. Use arrow keys to play the game:
   - Left/Right arrows: Move Ben
   - Up arrow: Jump
   - Down arrow: Fast fall
   - Collect stars and avoid peas!
